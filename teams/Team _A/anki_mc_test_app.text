Was ist die Hauptaufgabe der Datei app.py?	Die Datei steuert den Ablauf der Streamlit-App, lädt Konfiguration und Fragen, verwaltet Benutzer und rendert die passenden Ansichten (Start, Frage, Ende, Admin).
Welche Hauptbibliothek wird für die Benutzeroberfläche verwendet?	Streamlit (import streamlit as st).
Welche Funktion ist der Einstiegspunkt der Anwendung?	die Funktion main().
Wie startet man die Anwendung?	Mit dem Befehl: streamlit run app.py
Was macht der Abschnitt „Pfad-Setup für robuste Imports“?	Er fügt das Projektverzeichnis zum Python-Suchpfad hinzu, damit Module wie config oder auth korrekt importiert werden können.
Welche Umgebungsvariablen werden geladen?	Die Variablen aus der .env-Datei mittels load_dotenv().
Warum wird locale.setlocale() verwendet?	Um eine korrekte alphabetische Sortierung von Namen mit Umlauten zu gewährleisten.
Was passiert, wenn keine .env oder secrets.toml gefunden wird?	Eine Warnung wird in Streamlit angezeigt mit Hinweisen zur lokalen Installation.
Welche Aufgabe hat init_database()?	Sie initialisiert die Datenbank und erstellt ggf. Tabellen, falls sie fehlen.
Wie werden die Fragen geladen?	Durch load_questions() auf die in st.session_state ausgewählte Datei.
Wie wird der Benutzer authentifiziert?	Durch handle_user_session(), das die Login-Seite rendert oder eine user_id zurückgibt.
Wie erkennt die App, ob ein Nutzer Admin ist?	Durch is_admin_user(user_id, app_config).
Was passiert, wenn der Test beendet ist?	render_final_summary() zeigt die Ergebnisübersicht.
Wann wird render_question_view() ausgeführt?	Wenn eine aktuelle Frage angezeigt werden soll (nicht Admin, nicht fertig).
Was macht render_sidebar()?	Sie rendert die Seitenleiste mit Navigation und Optionen.
Was ist st.session_state.test_start_time?	Der Zeitpunkt, an dem der Test begonnen wurde.
Wie erkennt die App, dass sie lokal läuft?	Mit der Funktion is_request_from_localhost().
Wann wird das Admin-Panel angezeigt?	Wenn show_admin_panel in session_state True ist, der Nutzer Admin ist und der Aufruf lokal erfolgt.
Was passiert, wenn die Admin-Session ungültig ist?	Es erscheint eine Fehlermeldung, und der Nutzer muss sich neu einloggen.
Was macht render_final_summary()?	Es zeigt die Zusammenfassung der Testergebnisse und die Auswertung an.
Was passiert am Ende des Scripts?	Wenn die Datei direkt ausgeführt wird, ruft Python main() auf.
